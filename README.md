# MangaRender

源码放入\Engine\Programs路径下

选择MangaRender启动


## 操作方法

鼠标左键：按住可拖拽

鼠标滚轮：缩放

鼠标侧键（前进）：下一页

鼠标侧键（后退）：上一页

其他：

* 拖拽文件夹打开

## 最小程序集

```
MangaRender/
├── Engine/
│   ├── Binaries/
│   │   ├── Win64/
│   │   │   ├── MangaRender.exe // 启动
│   │   │   └── tbbmalloc.dll
│   ├── Content/
│   │   └── Slate/
│   └── Shaders/
        └── StandaloneRenderer/
```
